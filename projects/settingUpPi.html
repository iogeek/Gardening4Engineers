Setting up the Raspberry Pi

There are plenty of great setup tutorials. Here are the ones I use to setup the pi.

  • Enable Pi to boot from USB.
  • Pick the OS you plan to use from raspberrypi site..
    • If you prefer headless for iot settings pick “Raspberry Pi OS Lite”.
  • Drop the OS on a flash drive. Several options for Windows. I started with Etcher (src on Github) but moved to Raspberry Pi Imager which works with both USB and SD media. Another option is WinFlashTool.

Headless tweaks

Poached from here

  • Step 1: Enable ssh from WSL Assuming the drive letter is e:
    sudo mkdir -p /mnt/e && sudo mount -t drvfs e: /mnt/e

touch /mnt/e/ssh

  • Step 2: Setup Wifi SSID and keys in /mnt/e/wpa_supplicant.conf add the following
    country=US
    ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev 
    update_config=1
    network={
        ssid="YourNetworkSSID"
        psk="Your Network's Passphrase"
        key_mgmt=WPA-PSK
    }

Remove USB drive and boot Pi

The defaults will be” system name -> raspberrypi username -> pi password -> raspberry


NOTE: Once you boot, be sure to change your password and lock down ssh access.
